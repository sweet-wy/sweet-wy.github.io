想对王陆羽说的一百句情话

发表于 2022-07-04
  • 王陆羽与甜风的私密空间
首页

# 1. 郁金香哪有你的浴巾香。

# 2. 我 180 度地躺，你 90 度地坐，搭一架，只有一二象限的坐标系

# 3. 每天只想和你做四件事，一日三餐。

# 4. 你神秘的耳语和呼吸，被我系上这一颗心，沉入瓶底，目的是，造一罐，不被后人理解的蜜。

# 5. 我的被子又香又软又舒服，要不要跟我一起盖。

# 6. 花有什么好种的，来和我种草莓。

# 7. 有些事不用在一晚上做完，我们又不赶时间 可以每晚都做一做

# 8. 想和你朝朝又暮暮，嗯嗯又啊啊。

# 9. 想在你身上钻木取火，点燃此生。

# 10. 想送你很多很多口红，让你每天还我一点。

# 11. 我们浪漫点好不好 怎么浪漫点？我浪点，你慢点

# 12. 取悦你的方式有 70 种，一是 shopping，剩下 69

# 13. 夜深人静时，偷偷想我的先生。

# 14. 你和段位，我都想上。

# 15. 你知道我想变成什么样的人儿吗？变成你的人。

# 16. 别老是梦见我，我会打喷嚏的。

# 17. 地铁知识标写：站稳扶好。今晚我也打算这样提醒你。

# 18. 过程我可以哭，但你不能停，不然就是你不行。

# 19. 我们吵一架吧，我让你看看，我是怎么顶撞你的。

# 20. 我心里装的怀里抱的身下压的都是你。

# 21. 虽然我高数不及格，但我能在两秒之内解开你。

# 22. 我是千你是北，我们合一起就是 “乖”。

# 23. 我是锄禾你是当午。

# 24. 别的人只想和你上床，我不一样，沙发、厨房、阳台都可以。

# 25. 说不上你哪里好，就是想看你洗澡。

# 26. 你像风来了又走 我的身体满了又空

# 27. 今晚吃什么？火锅吗？外卖吗？可以是我吗？

# 28. 爱你不是两三天 而是一次又一次。

# 29. 你喜欢田野，而我愚笨。只能植荒十年，换得一时春生。

# 30. 喝醉酒是一件很色情的事，因为满脑子都是你。
